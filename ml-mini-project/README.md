## Mini-project: Authorship Attribution of Iskra Texts

This repository contains a mini-project reproducing and extending
distance-based authorship attribution methods described in
O.V. Alieva's work on the Iskra newspaper corpus.

### Contents
- data/: training and dubia texts
- notebooks/: preprocessing, distance-based methods, ML models, analysis
- results/: tables and figures
- paper/: report describing the experiment
